# Orchestration

Sample notebooks demonstrating how to use popular orchestration frameworks with Gemini on Google Cloud.

## Notebooks

| Notebook | Description |
| --- | --- |
| [intro_langchain_gemini.ipynb](intro_langchain_gemini.ipynb) | Introduction to using LangChain with Gemini |
| [intro_langgraph_gemini.ipynb](intro_langgraph_gemini.ipynb) | Introduction to using LangGraph with Gemini |
| [langgraph_gemini_podcast.ipynb](langgraph_gemini_podcast.ipynb) | Build a podcast generation workflow with LangGraph and Gemini |
| [llamaindex_workflows.ipynb](llamaindex_workflows.ipynb) | LlamaIndex workflows with Gemini |
| [autogen_gemini.ipynb](autogen_gemini.ipynb) | Multi-agent orchestration with AutoGen and Gemini |

## Getting Started

1. Open one of the notebooks in [Google Colab](https://colab.research.google.com/) or [Vertex AI Workbench](https://cloud.google.com/vertex-ai/docs/workbench/introduction).
2. Follow the instructions in the notebook to set up your Google Cloud project.
3. Run the cells to explore orchestration patterns with Gemini.
