# Evaluation for livekit-adk

## Procedures

1. Open `livekit-adk` in a new window
2. Open terminal and run `uv sync` to install dependencies
3. Run `uvicorn app.main:app --reload` to start the server
4. Verify the application at `http://localhost:8000`
5. Test the shopping agent functionality.
6. Write a `test_log_<timestamp>.md` to `livekit-adk/tests` directory with the actual procedures, outcomes, errors or frictions, or possible improvement points for the entire process.
