# Long Context

Sample notebooks demonstrating the long context window capabilities of Gemini on Google Cloud.

## Notebooks

| Notebook | Description |
| --- | --- |
| [intro_long_context.ipynb](intro_long_context.ipynb) | Introduction to long context processing with Gemini |

## Getting Started

1. Open the notebook in [Google Colab](https://colab.research.google.com/) or [Vertex AI Workbench](https://cloud.google.com/vertex-ai/docs/workbench/introduction).
2. Follow the instructions in the notebook to set up your Google Cloud project.
3. Run the cells to explore long context capabilities.
