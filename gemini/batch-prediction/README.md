# Batch Prediction

Sample notebooks demonstrating batch prediction with Gemini on Google Cloud.

## Notebooks

| Notebook | Description |
| --- | --- |
| [intro_batch_prediction.ipynb](intro_batch_prediction.ipynb) | Introduction to batch prediction with Gemini |
| [monitor_batch_prediction_gemini_api.ipynb](monitor_batch_prediction_gemini_api.ipynb) | Monitor batch prediction jobs using the Gemini API |

## Getting Started

1. Open one of the notebooks in [Google Colab](https://colab.research.google.com/) or [Vertex AI Workbench](https://cloud.google.com/vertex-ai/docs/workbench/introduction).
2. Follow the instructions in the notebook to set up your Google Cloud project.
3. Run the cells to explore batch prediction capabilities.
