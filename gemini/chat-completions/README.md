# Chat Completions API

Sample notebooks demonstrating the OpenAI-compatible Chat Completions API with Gemini on Google Cloud.

## Notebooks

| Notebook | Description |
| --- | --- |
| [intro_chat_completions_api.ipynb](intro_chat_completions_api.ipynb) | Introduction to the Chat Completions API with Gemini |

## Getting Started

1. Open the notebook in [Google Colab](https://colab.research.google.com/) or [Vertex AI Workbench](https://cloud.google.com/vertex-ai/docs/workbench/introduction).
2. Follow the instructions in the notebook to set up your Google Cloud project.
3. Run the cells to explore the Chat Completions API.
