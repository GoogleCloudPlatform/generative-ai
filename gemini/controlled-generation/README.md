# Controlled Generation

Sample notebooks demonstrating controlled generation (structured output) with Gemini on Google Cloud.

## Notebooks

| Notebook | Description |
| --- | --- |
| [intro_controlled_generation.ipynb](intro_controlled_generation.ipynb) | Introduction to controlled generation with Gemini |

## Getting Started

1. Open the notebook in [Google Colab](https://colab.research.google.com/) or [Vertex AI Workbench](https://cloud.google.com/vertex-ai/docs/workbench/introduction).
2. Follow the instructions in the notebook to set up your Google Cloud project.
3. Run the cells to explore controlled generation techniques.
