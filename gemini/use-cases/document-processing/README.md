# Document Processing

Sample notebooks demonstrating document processing with Gemini on Google Cloud.

## Notebooks

| Notebook | Description |
| --- | --- |
| [document_processing.ipynb](document_processing.ipynb) | Document processing with Gemini |
| [doc_parsing_with_llamaindex_and_llamaparse.ipynb](doc_parsing_with_llamaindex_and_llamaparse.ipynb) | Document parsing with LlamaIndex and LlamaParse |
| [patents_understanding.ipynb](patents_understanding.ipynb) | Patent document understanding with Gemini |

## Getting Started

1. Open one of the notebooks in [Google Colab](https://colab.research.google.com/) or [Vertex AI Workbench](https://cloud.google.com/vertex-ai/docs/workbench/introduction).
2. Follow the instructions in the notebook to set up your Google Cloud project.
3. Run the cells to explore document processing techniques.
